Iconify

This is a simple jQuery plugin which iconify the html elements based on the file extensions in their attributes.

Usage

Include the iconify.jquery.js file

<script src="iconify.jquery.js"></script>

Then initiate the plugin in your document

<script>
$(document).ready(function(){
    $("a").iconify();
});
</script>

Options

You can set the following options:

  • atr Optional href by default : Tell it which attribute of the element to look for.

  • dir Optional css/iconify-assets/ by default : Path to the additional file of iconify

  • theme Optional default by default : Which theme to be used for icons

Hope for more options in next versions.

Examples usage

To look for data-url attribute in <span> tags:

$("span").iconify({ atr : "data-url" });

If you want to use your own theme:

$("span").iconify({ atr : "data-url", theme : "myCustomTheme" });
